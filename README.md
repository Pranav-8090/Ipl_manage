# 🏏 IPL Management System

## 📌 Features
- Add Player
- Update Player
- Delete Player
- View Players
- Filter by Team & Category
- Top Scorer & Best Bowler

## ⚙️ Tech Used
- Java (JDBC)
- MySQL
- MVC Architecture

## 🚀 How to Run
1. Setup MySQL Database (ipl_db)
2. Add MySQL Connector JAR
3. Run Main Class
